# Sofon – AI Challenge Platform

## Problem Statement
AI practitioners and organizations often lack a focused, competitive environment to test and showcase AI solutions, while startups in this niche struggle to validate their product-market fit quickly.

## Solution Overview
We built Sofon, a full-stack AI challenge platform enabling organizations to host competitions and participants to engage in real-world AI problem solving. Designed with a 'Lean Startup' mindset, the platform incorporates early metric tracking and iterative improvements.

## Process Highlights

*   Defined brand identity, color palette, and visual language.
*   Led customer discovery efforts to secure the first client base.
*   Prepared investor-ready pitch materials and platform demos.
*   Applied Lean Startup metrics to guide feature prioritization and iteration.

# DCA Wallet – Self-Hosted Bitcoin Tracker

## Problem Statement
Bitcoin investors seeking to apply Dollar-Cost Averaging strategies often rely on centralized tools that compromise privacy and self-custody principles.

## Solution Overview
DCA Wallet is a minimalist, self-hosted, watch-only wallet that allows users to track their Bitcoin reserves and apply DCA strategies without sharing any data with third parties. The project is fully open source, emphasizing financial sovereignty.

## Process Highlights
*   Identified personal pain point and validated need through user interviews.
*   Designed brand identity, UI components, and user flows.
*   Developed backend in Python (FastAPI) and frontend in React + TypeScript + Tailwind.
*   Implemented MongoDB database and Dockerized the entire stack for easy self-hosting.
*   Led a small team of 3 developers, managing code reviews, sprints, and technical roadmap.
*   Prepared documentation and contribution guidelines to foster open-source adoption.

## Outcome Metrics
*   Core MVP features implemented: 100% completion
*   External contributors onboarded: 3 developers
